### ğŸ“˜ Deep Concepts Explained

#### ğŸ”¹ Linear Search

**Definition:** Sequentially check each element until the target is found.

**Algorithm Steps:**

1. Input: list `L` and target `x`
2. For each element in `L`, check if it equals `x`
3. If found, return index or True
4. If not found, return -1 or False

**Real-world analogy:** Searching for a book on a shelf by checking each one in order.

---

#### ğŸ”¹ Python Implementation: Linear Search

```python
def linear_search(lst, target):
    for i in range(len(lst)):
        if lst[i] == target:
            return i
    return -1

numbers = [5, 3, 8, 6, 7]
print("Index of 6:", linear_search(numbers, 6))  # Output: 3
print("Index of 10:", linear_search(numbers, 10))  # Output: -1
```

#### ğŸ”¹ Counting Occurrences

**Task:** Count how many times an element appears.

```python
numbers = [1,2,2,3,2,4,5]
count = 0
for num in numbers:
    if num == 2:
        count += 1

print("2 appears", count, "times")
```

**Using dictionary for multiple elements:**

```python
words = ["apple", "banana", "apple", "cherry"]
freq = {}
for word in words:
    if word in freq:
        freq[word] += 1
    else:
        freq[word] = 1

print(freq)  # {'apple': 2, 'banana': 1, 'cherry': 1}
```
