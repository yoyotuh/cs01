#### ğŸ” What is a Loop?

**Academic definition:**
A loop is a control structure that repeats a block of code until a condition is met.

**Beginner analogy:**
Loops are like alarm clocks that ring every morning until you turn them off â€” repetition until condition ends it.

---

### ğŸ§  Why Do We Need Loops?

Without loops:

```python
print("Hello")
print("Hello")
print("Hello")
print("Hello")
print("Hello")
```

With loop:

```python

    print("Hello")
```

> Loops = **write once, repeat many times**

### ğŸ§® `for` loop with `range()`

Range patterns:

| Pattern             | Meaning         |
| ------------------- | --------------- |
| `range(n)`          | 0 â†’ n-1         |
| `range(a, b)`       | a â†’ b-1         |
| `range(a, b, step)` | step increments |

```python

    print(i)   # prints 1 to 5
```

### ğŸ§· Accumulator Pattern (Very Important)

```python

for i in range(1, 6):
    total = total + i
print(total)   # 1+2+3+4+5 = 15
```

This pattern appears constantly in computing: sum, count, search, average, etc.
