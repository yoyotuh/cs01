### ğŸ“˜ Deep Concepts Explained

#### ğŸ”¹ Lists

* **Definition:** Ordered, mutable collection of items
* **Syntax:** `[item1, item2, item3]`
* **Access:** `list[index]` (0-based indexing)

```python
fruits = ["apple", "banana", "cherry"]
print(fruits[0])  # apple

fruits.append("date")  # add item
fruits[1] = "blueberry"  # modify item
```

---

#### ğŸ”¹ Tuples

* **Definition:** Ordered, immutable collection of items
* **Syntax:** `(item1, item2)`
* **Access:** Same as list, but cannot modify

```python
coordinates = (10, 20)
print(coordinates[1])  # 20
# coordinates[0] = 5 â†’ Error
```

**Use-case:** Fixed data like coordinates, RGB values, constants.

---

#### ğŸ”¹ Strings as Sequences

* Strings are **immutable sequences of characters**
* Access by indexing or slicing

```python
text = "Hello World"
print(text[0])      # H
print(text[0:5])    # Hello
print(text[-5:])    # World
```

**Useful string methods:**

* `len(text)` â€” length of string
* `text.upper()` â€” uppercase
* `text.lower()` â€” lowercase
* `text.replace("Hello","Hi")`
* `text.split()` â€” split into list by spaces
