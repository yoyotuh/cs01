#### üîπ Capstone Project Overview

**Project Idea:**
**‚ÄúStudent Performance Manager‚Äù**

Features:

1. Add, update, and delete students
2. Search for a student by name (linear or binary search)
3. Count students above/below a grade threshold
4. Display statistics (highest, lowest, average)
5. Sort students by grade (bubble sort / merge sort)
6. Recursive features (e.g., calculating factorial, sum, or nested list sums)
7. Handle invalid input and exceptions

**Design Steps:**

1. **Plan modules:** separate functions for each feature
2. **Choose data structures:** list of dictionaries for student records
3. **Implement algorithms:** search, count, sort, recursion
4. **Test extensively:** small datasets first
5. **Add input validation:** ensure program doesn‚Äôt crash
6. **Optimize performance:** avoid redundant loops, use dictionaries where needed

### ‚úÖ Python Code Example ‚Äî Capstone Skeleton

```python
students = []

def add_student(name, grade):
    students.append({"name": name, "grade": grade})

def update_student(name, grade):
    for s in students:
        if s["name"].lower() == name.lower():
            s["grade"] = grade
            return True
    return False

def delete_student(name):
    global students
    students = [s for s in students if s["name"].lower() != name.lower()]

def search_student(name):
    for s in students:
        if s["name"].lower() == name.lower():
            return s
    return None

def count_above(threshold):
    return sum(1 for s in students if s["grade"] > threshold)

def statistics():
    if not students:
        return None
    grades = [s["grade"] for s in students]
    return {"max": max(grades), "min": min(grades), "average": sum(grades)/len(grades)}

# Example: Recursive function for factorial
def factorial(n):
    if n == 0: return 1
    return n * factorial(n-1)

# Sorting: Bubble Sort by grade
def sort_students_by_grade():
    n = len(students)
    for i in range(n):
        for j in range(0, n-i-1):
            if students[j]["grade"] > students[j+1]["grade"]:
                students[j], students[j+1] = students[j+1], students[j]

# Sample usage
add_student("Alice", 90)
add_student("Bob", 75)
add_student("Charlie", 85)

print("Student Stats:", statistics())
print("Students above 80:", count_above(80))
sort_students_by_grade()
print("Sorted Students:", students)
```

**Exercise 2 ‚Äî Recursive Sum of Grades**

```python
def recursive_sum_grades(lst):
    if not lst:
        return 0
    return lst[0]["grade"] + recursive_sum_grades(lst[1:])

print("Total Grades:", recursive_sum_grades(students))
```
