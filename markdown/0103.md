### üìò Deep Concepts Explained

#### üßÆ Expressions & Operators

An **expression** produces a value.
Operators define **how values combine**.

| Category            | Operators         | Meaning            |
| ------------------- | ----------------- | ------------------ |
| Arithmetic          | `+ - * / // % **` | Math operations    |
| Comparison          | `== != > < >= <=` | True/False checks  |
| Logical             | `and or not`      | Combine conditions |
| Assignment          | `=`               | Store value        |
| Compound Assignment | `+= -= *= /=`     | Update variable    |

**Example expression:**

```python
x = (10 + 5) * 2
```

---

#### ‚öñÔ∏è Order of Evaluation (PEMDAS in Python)

1. Parentheses `()`
2. Exponent `**`
3. Multiplication, Division, Floor `*, /, //, %`
4. Addition, Subtraction `+, -`
5. Comparison `>, <, ==, ...`
6. Logical `and`, `or`, `not`

Example:

```python
result = (4 + 3) * 2 > 10 and not False
```

Breakdown:

* `(4 + 3) = 7`
* `7 * 2 = 14`
* `14 > 10` = True
* `not False` = True
* `True and True` = True

‚úÖ Program thinking, not guessing.

### üîÄ Control Flow: if / elif / else

Programs adjust behavior based on conditions.

```python
age = int(input("Enter age: "))

if age >= 18:
    print("Adult")
elif age >= 13:
    print("Teenager")
else:
    print("Child")
```

> Computers think in **conditions and consequences**.

### üè´ In-Class Exercises (3)

#### Exercise 1

Ask for a number, print whether it's **positive, negative, or zero**.

#### Exercise 2

Input three numbers, print the **largest**.

#### Exercise 3

Student attendance system:
If attendance >= 75% ‚Üí "Allowed for exam"
Else ‚Üí "Not allowed"

---

### ‚úÖ Solutions

#### Solution 1

```python
num = float(input("Enter number: "))

if num > 0:
    print("Positive")
elif num < 0:
    print("Negative")
else:
    print("Zero")
```

#### Solution 2

```python
a = float(input("A: "))
b = float(input("B: "))
c = float(input("C: "))

if a >= b and a >= c:
    print(a, "is largest")
elif b >= a and b >= c:
    print(b, "is largest")
else:
    print(c, "is largest")
```

#### Solution 3

```python
attendance = float(input("Attendance %: "))

if attendance >= 75:
    print("Allowed for exam")
else:
    print("Not allowed")
```
