#### ğŸ”¹ What is an Algorithm?

**Definition:** A finite sequence of steps to solve a problem.

**Real-world analogy:**

* Cooking a recipe â†’ steps, order, input ingredients â†’ final dish
* Algorithm = Recipe, Program = Cooked Dish

**Properties of algorithms:**

1. Finiteness â†’ terminates after finite steps
2. Definiteness â†’ each step is clear and unambiguous
3. Input â†’ zero or more values
4. Output â†’ produces at least one result
5. Effectiveness â†’ steps are basic and executable

---

#### ğŸ”¹ Pseudocode

* Plain-English, step-by-step representation of an algorithm
* Not language-specific, focuses on **logic**

**Example:** Sum first N numbers

```
Algorithm SumFirstN
Input: integer N
Output: sum of numbers 1 to N

1. Initialize sum = 0
2. For i = 1 to N
     sum = sum + i
3. Output sum
```
