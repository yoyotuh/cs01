#### ğŸ’¾ What is Data?

**Academic definition:**
Data is raw information that a program stores, manipulates, and processes to produce meaningful output.

**Beginner analogy:**
Data is like ingredients in cooking.
Your program is a kitchen that turns ingredients (data) into dishes (results).

---

#### ğŸ“¦ What is a Variable?

**Academic definition:**
A variable is a named memory location used to store data.

**Beginner analogy:**
A variable is like a labeled container. You store something inside and use its label to find it later.

Example:

| Variable Name | Stored Value |
| ------------- | ------------ |
| age           | 21           |
| name          | "Sarah"      |
| is_student    | True         |
