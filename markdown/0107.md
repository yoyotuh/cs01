### üìò Deep Concepts Explained

#### üîπ Dictionaries (dict)

* **Definition:** Unordered, mutable collection of key-value pairs
* **Syntax:** `{key1: value1, key2: value2}`
* **Access:** `dict[key]`

```python
student = {"name": "Alice", "age": 20, "major": "CS"}
print(student["name"])  # Alice
```

**Add/Update:**

```python
student["year"] = 2
student["age"] = 21
```

**Delete:**

```python
del student["major"]
```

**Iterate:**

```python
for key, value in student.items():
    print(key, ":", value)
```

---

#### üîπ Sets

* **Definition:** Unordered collection of **unique** elements
* **Syntax:** `{element1, element2}`

```python
numbers = {1,2,3,3,4}
print(numbers)  # {1, 2, 3, 4} ‚Üí duplicates removed
```

**Set Operations:**

```python
A = {1,2,3}
B = {2,3,4}

print(A.union(B))        # {1,2,3,4}
print(A.intersection(B)) # {2,3}
print(A.difference(B))   # {1}
```

**Use-case:** Removing duplicates, membership checks, math operations

### ‚úÖ Python Code Examples

#### Example 1 ‚Äî Dictionary Operations

```python
person = {"name": "Bob", "age": 25}
person["city"] = "Jakarta"    # Add
person["age"] = 26            # Update
print(person)

for key, val in person.items():
    print(f"{key} -> {val}")
```

#### Example 2 ‚Äî Set Operations

```python
fruits1 = {"apple", "banana", "cherry"}
fruits2 = {"banana", "kiwi", "apple"}

print("Union:", fruits1 | fruits2)
print("Intersection:", fruits1 & fruits2)
print("Difference:", fruits1 - fruits2)
```

#### Example 3 ‚Äî Using Dictionaries for Counting

```python
text = "hello world"
count = {}

for char in text:
    if char in count:
        count[char] += 1
    else:
        count[char] = 1

print(count)
```

### ‚ö†Ô∏è Common Mistakes & Best Practices

| Mistake                               | Solution                                        |
| ------------------------------------- | ----------------------------------------------- |
| Accessing non-existent dictionary key | Use `dict.get(key)` or check `if key in dict:`  |
| Assuming order in sets                | Sets are unordered; use `sorted()` if needed    |
| Modifying set while iterating         | Avoid; iterate over a copy or convert to list   |
| Counting without init                 | Initialize dictionary or use `defaultdict(int)` |
